How to Play

To complete your quest, you need to edit keyquest.js retrieve various magical items from the inhabitants of Winderos. Fortunately, all the inhabitants of Winderos speak HTTP+JSON.

You can see a list of all the available characters at /help

To interact with one of the characters, you need to make an HTTP POST to their endpoint. Some interactions require you to use a magical item, or offer them a piece of treasure.

Here's how you would offer the Talisman of Optimisation to the cleric Baliskov:

POST /cleric HTTP/1.1
{ "name" : "Talisman of Optimisation" }

If your encounter is successful, they'll return a 200 OK and include a magical item in JSON format. If your encounter is not successful, they'll return an HTTP error status and your quest is over.

Workshop Objectives

  1. Use JavaScript Promises and create a promise chain to get the Control Key from the evil wizard Ballmerack.
  2. Once you've done that, create promise chains for retrieving the Alt Key and the Delete Key
  3. Use the Promise.all([a,b,c]) method to run some code when you've retrieved all three keys.
  4. Bonus points: add an extra character and encounter to each of your three quests.
